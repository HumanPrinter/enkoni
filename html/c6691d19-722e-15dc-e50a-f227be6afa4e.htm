ShallowDatabaseRepositoryTEntity ClassEnkoni Framework Class Library
This class extends the DatabaseRepositoryTEntity class and implements some of the functionality using the Entity Framework. The repository can be used for types that only need to be created, but not saved or retrieved as this is done by the parent-type's repository.
