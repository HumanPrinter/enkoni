FileRepositoryTEntityResumeSourceFileMonitor Method Enkoni Framework Class Library
Resumes the monitor that monitors the source file. This method is added as a counterpart for the PauseSourceFileMonitor method, however since the file monitor does not have any specific pause/resume behavior, calling this method is the same as calling StartSourceFileMonitor.
