Comparer T  ClassEnkoni Framework Class Library
Provides a generic implementation of IComparer T  that is able to compare two objects based on a specified field name, which can even be concatenated using the '.' character. The final field by which the objects must be compared must implement the CompareTo method which is defined by the IComparable and IComparable T  interfaces.
