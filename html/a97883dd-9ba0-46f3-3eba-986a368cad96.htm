DutchPhoneNumberAreaCodeCollection
Defines the configuration element collection that holds the area codes that should either be added or removed from the list of area codes that must be validated by the DutchPhoneNumberAttribute.
DutchPhoneNumberAreaCodeConfigElement
Defines the configuration element that holds a single area code that should either be added or removed from the list of area codes that must be validated by the DutchPhoneNumberAttribute.
DutchPhoneNumberValidationConfigDictionary
Defines a custom dictionary that holds the DutchPhoneNumberValidationConfigElement that were read from the configuration file.
DutchPhoneNumberValidationConfigElement
Defines the configuration element that can be used to configure the DutchPhoneNumberAttribute.
