CircularStackT
Represents a fixed size last-in-first-out (LIFO) collection of instances of the same arbitrary type that uses a circular collection as internal storage.
ComparerT
Provides a generic implementation of IComparerT that is able to compare two objects based on a specified field name, which can even be concatenated using the '.' character. The final field by which the objects must be compared must implement the CompareTo method which is defined by the IComparable and IComparableT interfaces.

LambdaEqualityComparerT, TField
This class compares types based on a field of the type. By using this class, it is no longer required to create a specific equality comparer to compare types using just one field.
