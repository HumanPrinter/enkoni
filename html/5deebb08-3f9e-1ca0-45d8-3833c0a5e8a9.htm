A signed number indicating the relative values of the two numbers.
Return Value Description
Less than zero: x is less than y -or- x is not a number (NaN) and y is a number.
Zero: x is equal to y -or- Both x and y are not a number (NaN), PositiveInfinity, or NegativeInfinity.
Greater than zero: x is greater than y -or- x is a number and y is not a number (NaN).
