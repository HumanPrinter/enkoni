The Extensions type exposes the following members.
Methods
  NameDescription
Fire
Fires an event in a sequential way. An event handler needs to finish before the next event handler will be called. This method will return when all the event handlers have finished.
This method will automatically check if there are any event handlers subscribed to handler and it will automatically propagate the call the the appropriate thread.
FireAsync
Fires an event in an asynchronous way. When this method returns, the event handlers may still be running.
This method will automatically check if there are any event handlers subscribed to handler and it will automatically propagate the call the the appropriate thread.
FireInParallel
Fires an event in a parallel way. This method will return when all the event handlers have finished.
This method will automatically check if there are any event handlers subscribed to handler and it will automatically propagate the call the the appropriate thread.
