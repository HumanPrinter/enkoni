DutchPhoneNumberAreaCodeCollection
Defines the configuration element collection that holds the area codes that should either be added or removed from the list of area codes that must be validated by the DutchPhoneNumberValidator.
DutchPhoneNumberAreaCodeConfigElement
Defines the configuration element that holds a single area code that should either be added or removed from the list of area codes that must be validated by the DutchPhoneNumberValidator.
DutchPhoneNumberValidatorConfigDictionary
Defines a custom dictionary that holds the DutchPhoneNumberValidatorConfigElement that were read from the configuration file.
DutchPhoneNumberValidatorConfigElement
Defines the configuration element that can be used to configure the DutchPhoneNumberValidator.
EmailDomainCollection
Defines the configuration element collection that holds the domains that should either be included or excluded from the list of domains that must be validated by the EmailValidator.
EmailDomainConfigElement
Defines the configuration element that holds a single domain that should either be included or excluded from the list of domains that must be validated by the EmailValidator.
EmailValidatorConfigDictionary
