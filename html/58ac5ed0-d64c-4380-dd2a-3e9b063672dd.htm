ApplyIdentifiers(IEnumerableTEntity)
Applies new identifiers to the entities starting with identifier '1'. More often then not, entities that are read from a file do not have any identifiers. Therefore, they are applied here. If the source file already specifies identifiers for each record, override this method with an empty implementation to disable this behavior.
ApplyIdentifiers(IEnumerableTEntity, Int32)
Applies new identifiers to the entities starting with the specified start value. More often then not, entities that are read from a file do not have any identifiers. Therefore, they are applied here. If the source file already specifies identifiers for each record, override this method with an empty implementation to disable this behavior.
