DatabaseRepositoryTEntityAddEntitiesCore Method Enkoni Framework Class Library
Adds a collection of new entities to the repository. They are added to the addition cache until it is saved using the SaveChanges method. A temporary (negative) RecordID is assigned to the entities. This will be reset when the entity is saved.
