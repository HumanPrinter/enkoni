comparisonFactor
Type: System Double
The factor that must be taken into account. If compareOption is set to 
            Margin, the comparison factor will be treated as an absolute margin. If compareOption is
            set to SignificantDigits the comparison factor will be treated as the number of digits that must be 
            examined will comparing. Note that the comparison factor in that case will be truncated to an integer.
compareOption
Type: Enkoni.Framework DoubleCompareOption
Defines the method that must be used to compare the double values.
Return Value
Type: Int32
A signed number indicating the relative values of the two numbers. 

            Return Value Description 

            Less than zero: source is less than value -or- source is not a number 
            (NaN) and value is a number.

            Zero: source is equal to value -or- Both source and value are 
            not a number (NaN), PositiveInfinity, or NegativeInfinity.

            Greater than zero: source is greater than value -or- source is a number and 
            value is not a number (NaN).
