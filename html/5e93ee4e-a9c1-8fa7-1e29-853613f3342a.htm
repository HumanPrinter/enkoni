Mvc3FixedModelBinder Class

Provides an override for DefaultModelBinder in order to implement fixes to its behavior. The MVC3 library from Microsoft contains a couple of bugs that cause runtime exceptions when binding received data to the model. The default translation of an enum from server to client is enum-to-int. However, the default translation of an enum from client to server is string-to-enum. This bug has been fixed in MVC4.
