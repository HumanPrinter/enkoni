Enkoni.Framework.Logging NamespaceEnkoni Framework Class Library
Contains functionality to add logging to an application using the Microsoft Enterprise Library's Logging Application Block
Classes
  ClassDescription
Logger
This class sends log messages using the Logging Application Block which is part of the Microsoft Enterprise Library. An instance of this class can be retrieved through the LogManager class.
LogManager
Manages and gives access to a Logger instance that is capable of logging messages through the Logging Application Block which is part of the Microsoft Enterprise Library.
