EnumerablePartitionTKey, TSource Method (IEnumerableTSource, FuncTSource, TKey, IEqualityComparerTKey)

Partitions a collection into groups based on a key value. This methods differs from the default GroupByTSource, TKey(IEnumerableTSource, FuncTSource, TKey) method in the sense that it will create groups of elements with the same key, only if those elements are adjacent.
