Enkoni.Framework.Entities NamespaceEnkoni Framework Class Library
Contains functionality of a generic data-access layer which can be utilized in a typical multi-tier application.
Classes
  ClassDescription
CsvFileRepositoryTEntity
This class extends the abstract FileRepositoryTEntity class and implements the functionality to read and write from or to a CSV-file.
DatabaseRepositoryTEntity
This abstract class extends the abstract RepositoryT class and implements some of the functionality using the Entity Framework.
DatabaseSourceInfo
This class can be used by the DatabaseRepositoryTEntity to retrieve valuable information about the database that is to be used. This class is added for improved usability of the DataSourceInfo in combination with the DatabaseRepository.
DataSourceInfo
This class contains information about a specific data source. This can be any kind of data source. This class is utilized by the various repositories. Check the documentation of the used repository for more information about the required or supported data source-information.
DomainModel Obsolete.
This class defines the public API of a class that represents a domain model.
EntityT
Provides a basic implementation of an entity-type.
EntityEqualityComparerTEntity
This class can be used to compare two instances of IEntityT for equality. It uses the RecordId property to compare the two instances.
EntityValidatorT
This abstract class defines the public API of a class that can validate entities.
FileRepositoryTEntity
This abstract class extends the abstract RepositoryT class and implements some of the functionality using basic file I/O. This implementation can be used a base for any file format-specific file repositories.
FileSourceInfo
This class can be used by the FileRepositoryTEntity or any of its descendants to retrieve valuable information about the file that is to be used. This class is added for improved usability of the DataSourceInfo in combination with the FileRepository.
HttpSessionMemoryStoreT
This class implements the abstract class MemoryStoreT using collections that are stored in a HTTP Session.
InstanceMemoryStoreT
This class implements the abstract class MemoryStoreT using a simple collection variable.
MemoryRepositoryTEntity
This abstract class extends the abstract RepositoryT class and implements some of the functionality using memory storage.
MemorySourceInfo
This class contains static members that would normally be part of the MemorySourceInfoT class, but since that class is generic the static members are placed in this non-generic counterpart to avoid possible confusion about the use of the methods.
MemorySourceInfoT
This class can be used by the MemoryRepositoryTEntity to retrieve valuable information about the data store that is to be used. This class is added for improved usability of the DataSourceInfo in combination with the MemoryRepository.
MemoryStoreT
This abstract class defines the API of a class that holds a collection of objects in memory.
RepositoryT
This abstract class defines the API of a repository that is capable of accessing specific types in a persistency.
ServiceRepositoryTEntity
This abstract class extends the abstract RepositoryT class and implements some of the functionality using WCF data communication. This implementation can be used a base for any WCF-service repositories.
ServiceSourceInfo
This class can be used by the ServiceRepositoryTEntity or any of its descendants to retrieve valuable information about the service that is to be used. This class is added for improved usability of the DataSourceInfo in combination with the ServiceRepository.
ShallowDatabaseRepositoryTEntity
This class extends the DatabaseRepositoryTEntity class and implements some of the functionality using the Entity Framework. The repository can be used for types that only need to be created, but not saved or retrieved as this is done by the parent-type's repository.
StaticMemoryStoreT
This class implements the abstract class MemoryStoreT using a simple static collection variable.
SubDomainModelT
This abstract class defines the public API of a class that represents the sub domain of the domain model.
XmlFileRepositoryTEntity
This class extends the abstract FileRepositoryTEntity class and implements the functionality to read and write from or to an XML-file.
