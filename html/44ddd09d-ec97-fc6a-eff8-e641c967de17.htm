Enkoni.Framework.Entities NamespaceEnkoni Framework Class Library
Contains functionality of a generic data-access layer which can be utilized in a typical multi-tier application.
Classes
  ClassDescription
CsvFileRepository TEntity 
This class extends the abstract FileRepository TEntity  class and implements the functionality to read and write from or to a CSV-file.
DatabaseRepository TEntity 
This abstract class extends the abstract Repository T  class and implements some of the functionality using the Entity Framework.
DatabaseSourceInfo
This class can be used by the DatabaseRepository TEntity  to retrieve valuable information about the database that is to be used. This class is added for improved usability of the DataSourceInfo in combination with the DatabaseRepository.
DataSourceInfo
This class contains information about a specific data source. This can be any kind of data source. This class is utilized by the various repositories. Check the documentation of the used repository for more information about the required or supported data source-information.
DomainModel
This class defines the public API of a class that represents a domain model.
Entity T 
Provides a basic implementation of an entity-type.
EntityEqualityComparer TEntity 
This class can be used to compare two instances of IEntity T  for equality. It uses the RecordId property to compare the two instances.
EntityValidator T 
This abstract class defines the public API of a class that can validate entities.
FileRepository TEntity 
This abstract class extends the abstract Repository T  class and implements some of the functionality using basic file I/O. This implementation can be used a base for any file format-specific file repositories.
FileSourceInfo
This class can be used by the FileRepository TEntity  or any of its descendants to retrieve valuable information about the file that is to be used. This class is added for improved usability of the DataSourceInfo in combination with the FileRepository.
HttpSessionMemoryStore T 
This class implements the abstract class MemoryStore T  using collections that are stored in a HTTP Session.
MemoryRepository TEntity 
This abstract class extends the abstract Repository T  class and implements some of the functionality using memory storage.
MemorySourceInfo
This class contains static members that would normally be part of the MemorySourceInfo T  class, but since that class is generic the static members are placed in this non-generic counterpart to avoid possible confusion about the use of the methods.
MemorySourceInfo T 
This class can be used by the MemoryRepository TEntity  to retrieve valuable information about the data store that is to be used. This class is added for improved usability of the DataSourceInfo in combination with the MemoryRepository.
MemoryStore T 
This abstract class defines the API of a class that holds a collection of objects in memory.
Repository T 
This abstract class defines the API of a repository that is capable of accessing specific types in a persistency.
ServiceRepository TEntity 
This abstract class extends the abstract Repository T  class and implements some of the functionality using WCF data communication. This implementation can be used a base for any WCF-service repositories.
ServiceSourceInfo
This class can be used by the ServiceRepository TEntity  or any of its descendants to retrieve valuable information about the service that is to be used. This class is added for improved usability of the DataSourceInfo in combination with the ServiceRepository.
ShallowDatabaseRepository TEntity 
This class extends the DatabaseRepository TEntity  class and implements some of the functionality using the Entity Framework. The repository can be used for types that only need to be created, but not saved or retrieved as this is done by the parent-type's repository.
StaticMemoryStore T 
This class implements the abstract class MemoryStore T  using a simple static collection variable.
SubDomainModel T 
This abstract class defines the public API of a class that represents the sub domain of the domain model.
XmlFileRepository TEntity 
This class extends the abstract FileRepository TEntity  class and implements the functionality to read and write from or to an XML-file.
