Extensions SafeClose Method Enkoni Framework Class Library
Tries to gracefully close the communication object with regards for the current state of the communications object. If this fails due to a CommunicationException or a TimeoutException, it aborts the connection. If another exception is raised during the closing of the communication channel, the connection is still aborted but after the channel has been aborted the exception is rethrown.
